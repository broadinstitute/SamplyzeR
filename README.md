## Requirements
* R >= 3.3.2
* class (>= 7.3-14),
* ggplot2 (>= 2.2.1),
* WriteXLS (>= 4.0.0),
* grid (>= 3.3.2),
* gridExtra (>= 2.2.1)

## Installation
This tool is still under development, and haven't yet released to CRAN. To install from github, you will need to have `devtools` installed.

```r
library(devtools)
install_github('xiaolicbs/samplyzer')
```
